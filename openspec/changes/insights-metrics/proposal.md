## 为什么

业界成熟的自进化 AI Agent 系统提供完整的洞察/指标系统，分析会话历史生成使用情况报告，包含 token 消耗、成本估算、工具使用模式、活动趋势、模型/平台分解。NanoHermes 需要实现相同的洞察引擎。

## 变更内容

- 实现 InsightsEngine 类，查询会话数据生成报告
- 实现 token 消耗聚合和成本估算
- 实现工具使用模式提取和排名
- 实现每日活动趋势计算
- 实现终端输出格式化（条形图可视化）

## 能力

### 新增能力

- `insights-engine`: 洞察引擎，查询会话数据生成完整报告。包含概览、模型分解、平台分解、工具使用、技能使用、活动趋势、顶部会话。
- `cost-estimation`: 成本估算，基于 token 计数（input、output、cache read、cache write）和模型定价数据计算 USD 成本估算。
- `activity-trends`: 活动趋势，计算每日活动指标（会话数、消息数、token 数、成本），支持条形图可视化。

# 南京大学毕业致辞：守护好那些看似无用的热爱

> **来源**：抖音（抖音号：923091693）  
> **场景**：南京大学毕业典礼致辞节选

---

## 原文

接下来，我恳请大家：**守护好那些看似无用的热爱。**

毕业之后，你们会迅速变得越来越"有用"。你们会成为解决难题的能手、高效完成任务的高手、创造经济价值的行家里手。

但千万不要丢掉那些无功利的热爱：一首让你心头震颤的小诗，一段打动灵魂的乐曲，一本和专业毫不相干的闲书，一个不追求回报的小爱好。

正是这些纯粹、不带功利目的的爱好，区分开了你与流水线批量生产的产品。它们构筑起你的精神家园，当你被现实磋磨得灰头土脸时，这里就是你可以躲进去喘口气的心灵港湾。

人生不必事事计较收益，总要保留一点不计回报的温柔。

同学们，南京大学教会你们的，从来不止是公式、定律、理论与模型。更有对真理的敬畏，对生活的热忱，以及善待自我的从容。

愿你们心怀山海，也珍惜眼前的一饭一蔬；愿多年之后再重逢，你们依旧意气风发，面色红润，眼里有光。

## 核心观点提炼

| 观点 | 说明 |
|------|------|
| **无用的热爱** | 不追求功利回报的爱好，是区分人与机器的关键 |
| **精神家园** | 当现实压力袭来时，这些纯粹爱好是心灵的避风港 |
| **教育的本质** | 大学教的不仅是知识技能，更是对真理的敬畏、对生活的热忱、善待自我的从容 |
| **人生建议** | 不必事事计较收益，保留一点不计回报的温柔 |

## 与 AI 时代的呼应

这段毕业致辞虽然讲的是人生哲理，但放在 AI 时代有特别的意味：

| 传统观念           | AI 时代的新解读            |
| -------------- | -------------------- |
| "有用" = 能解决实际问题 | AI 已经比人更能解决实际问题      |
| "无用" = 不产生经济价值 | "无用"的热爱恰恰是人的独特性所在    |
| 教育 = 传授技能      | 教育 = 培养 AI 无法替代的人文素养 |
| 效率 = 价值        | 温柔、诗意、从容 = 人之为人的底色   |

**Claude Code 自审不靠谱的问题**，本质上也是"太有用"导致的盲区——模型专注于完成目标任务，却忽略了边界条件和异常场景。而人类审查者的价值，恰恰在于那些"不功利"的视角：直觉、经验、对细节的敏感、对不确定性的容忍。

**保留一点"无用"的热爱**，在技术领域或许就是：偶尔读一首诗、听一段音乐、翻一本和专业无关的书。这些看似无用的输入，反而可能成为你突破技术瓶颈的灵感来源。
